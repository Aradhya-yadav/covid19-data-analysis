# covid19-data-analysis
COVID-19 &amp; Vaccination Data Analysis using Python
Project Overview

This project performs Exploratory Data Analysis (EDA) on real-world COVID-19 and Vaccination datasets of India using Python.
The objective was to understand infection trends, state-wise impact, recovery and death rates, and vaccination distribution through data cleaning, feature engineering, and visualization.
